## ğŸ”® Project Introduction

Tarot Fate is a blockchain-based tarot card divination game that combines Web3 technology with traditional tarot culture. Players can participate through card drawing, synthesis, staking, and earn NFT rewards and token dividends.

### âœ¨ Core Features

- **ğŸ´ AI Divination System**: Pay with BNB or use tickets to draw single or triple cards, obtain random tarot cards
- **ğŸ¯ Card Rarity**: Includes Common, Rare, Hero, and Legend level cards
- **âš—ï¸ Card Synthesis**: Combine lower-tier cards into higher rarity cards
- **ğŸ”¥ Burn System**: Burn 3 Common cards to get 1 divination ticket
- **ğŸ’ Appraisal System**: Appraise cards to increase scores and earn more dividends
- **ğŸ–¼ï¸ NFT Minting**: Mint high-value cards as on-chain NFTs
- **ğŸ’° Staking Pool**: Stake FS (FateStar) tokens to earn BNB dividends
- **ğŸº FS Jar**: 20% of each BNB draw goes to buyback FS tokens into the jar, automatically distributed to holders when new Legend NFTs are minted
- **ğŸ¤ Invite System**: Invite friends to participate and earn rewards
- **ğŸ” Wallet Connection**: Support multiple Web3 wallets via RainbowKit

### ğŸ› ï¸ Tech Stack

- **Frontend Framework**: [Next.js 16](https://nextjs.org/) - React server-side rendering framework
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) - Utility-first CSS framework
- **Web3 Integration**:
  - [Wagmi](https://wagmi.sh/) - React Hooks for Ethereum
  - [Viem](https://viem.sh/) - TypeScript Interface for Ethereum
  - [RainbowKit](https://www.rainbowkit.com/) - Wallet connection UI components
  - [Ethers.js](https://docs.ethers.org/) - Ethereum interaction library
- **Blockchain Networks**: BSC (Binance Smart Chain) mainnet and testnet
- **3D Effects**: [OGL](https://oframe.github.io/ogl/) - Lightweight WebGL library
- **HTTP Client**: [Axios](https://axios-http.com/)
- **Development Tools**: ESLint, PostCSS
